## Linux

Membuat folder baru :

```bash
mkdir [NAMA_FOLDER]
```

Membuat file baru :

```bash
touch [NAMA_FILE.JENIS_FILE]
```

Menampilkan list :

```bash
ls -l /
```

Note :

1. `-l` digunakan untuk menampilkan "long", yaitu informasi metadata yang detail.
2. `/` merupakan root directory.

Menghapus secara paksa :

```bash
rm -f
```

Note :

1. `rm` berarti remove.
2. `-f` berarti force.

## Windows

Membersihkan Command Prompt :

```bash
cls
```

### BOTH

Mentransfer data ke atau dari sebuah server menggunakan berbagai network protocols :

```bash
curl [OPTION] [URL]
```

Note :

1. `[OPTION]` diisi dengan opsi / configurasi tambahan.
2. `[URL]` diisi dengan link URL yang ingin di akses.
